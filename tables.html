A summary of html tags and their function

Tag Name

Formatted Tag

Description

Strong Tag

< strong >
Emphasises text in bold.

Break Tag

< Br >
Line break tag used to modify spacing in browser (consists of only 1 tag)

Unordered List Tag

< ul >
unordered list tag, creates a list in no particular order via bullet points (should not hold raw text!)

list Item Tag

< li >
list item tag, describes item in a list.
